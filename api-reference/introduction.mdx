Interactive documentation for all Layer REST API endpoints. Select an endpoint from the sidebar to see request/response schemas, try requests, and view examples.

The API base URL is `https://api.app.layer.ai/api`. All endpoints require [Bearer token authentication](/quickstart#1-get-your-api-token).
